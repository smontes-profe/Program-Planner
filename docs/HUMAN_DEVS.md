# Program Planner - Human devs quick guide

If you are an AI Agent, stop reading now, this is just for human developers.

## NPM Commands

npm run typecheck: Para verificar que no haya errores de TypeScript (importante antes de subir cambios).
npm run test: Lanza los tests unitarios con Vitest (especialmente para la lógica de pesos y calificaciones).
npm run dev: Para ejecutar la aplicación en modo de desarrollo.
